# Default-HWR
Default Healthcare Worker Registry

## Setup
1. Edit the following files:
    * ```nginx/default-hwr-nginx```
    * ```nginx/default-hwr-nginx.conf```
    * ```upstart/default-hwr-upstart.conf```
    * ```config.py```
    * ```install.sh```
2. Run ```install.sh```
    ```
    ./install.sh
    ```
